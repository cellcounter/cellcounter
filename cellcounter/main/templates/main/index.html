Haematocritters provides a simple way for cells on a blood film to be counted and analysed. Cell types can be input with one hand as they are identified, and pressing space+key allows abnormal cell types to be counted.

For ambiguous cells, pressing shift+key shows normal examples, and shift+space+key shows abnormal examples.
