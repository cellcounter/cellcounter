Blood Count provides a simple way for cells on a blood film to be counted and analysed. as they are identified.

Input modeCell types can be input with one hand. Pressing space+key allows abnormal cells of each type to be selected.

View modeNormal examples can be viewed by pressing shift+key. Abnormal examples can be viewed by pressing shift+space+key.
